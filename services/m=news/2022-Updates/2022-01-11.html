Some Core Changes
  • Up to Legacy Update List
  • Refresh
  Summer
Website Developer
11-January-2022 00:08

After the December 15th updates of trying to make some changes to the core, the server decided to have a few stability issues. This update was to revert some of these changes to the core that made the game pretty unstable. As Ceikry stepped away for some much needed time away from the internet as a whole; Ryan became the Community Manager and System Administrator.

Authentic Changes

  • Fixed the incorrect message when trying to repair blast furnace without a hammer.
  • Can now travel on free charter boats without having coins in inventory.
  • Set the magic dart base xp to 30.
  • Can now pickpocket women in ardy.
  • Fishing Guild is able to be entered with a fishing skill boost.
  • Removed range strength and prayer bonus from god armour.

Bot Fixes

  • Fixed Pest Control bots and a number of other potential issues with various bot functionalities.
  • Fixed Adventure Bots pooling/stuck in Lumbridge after death.

Some Bug Fixes

  • Fixed a bug with canoes to where now you can take the canoe to the wild.
  • The Edgeville Dungeon gate no longer gets stuck.
  • Emerald and diamond bracelet enchants no longer produce noted items.
  • Prevented Blast Furnace temperature from going negative and deduplicate bar handling
  • Fixed the player saves for players who used coal on the conveyor belt before bf-fixes

The Core Changes that we talked about

  • Carefully reverted some core changes introduced after Dec 15 release, this should resolve current stability issues.
  • Removed the server monitor as it was blowing up in headless environments
