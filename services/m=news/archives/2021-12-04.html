Happy December!

The snow is back and the 2009scape Team has released the 2021 Christmas Event! It is called the 30 Days of Christmas. For the next 30 days, doing anything that gives experience in the game has chance of rewarding you with a Christmas present, which contains a bunch of amazing gifts!

About The Event

  • Players can collect up to 10 gifts a day
  • The gifts do no need to be opened immediately, they can be stockpiled.
  • Each individual action must be worth at least 20xp for it to trigger the chance at getting a gift.
  • There is a 5 second cool down between eligible XP drops to prevent exploiting.
If you would like to learn more check out the Christmas 2021 page!

We also have a few patch notes for you guys this update!

Authenticity Improvements

  • Random Event Genie Lamps now scale experience correctly
  • Animations for passing through Isafdar Dense Forest Added
  • Implement Wire Traps, Leaf Traps, and Stick Traps in Isafdar
  • Fairy rings emit a sound when teleporting
  • Void set effects now work more uniformly
  • Summoning familiars now grant combat xp when attacking
  • Canoe travel interface now animates correctly
  • Spiky vambraces crafting added

QoL Changes

  • The Launcher can now toggle the snow for December

Bug Fixes

  • Quest Point Cape/Hood no longer unequip on login if wear QP hood.
  • Void Mace now consumes runes correctly
  • Fixed how the attack skill cape perk is calculated. It should now have a much more noticeable effect
  • Made broad bolts tradeable and fix their ranged strength
  • Properly track leaf-bladed sword drops in ::stats command
  • Al Kharid Upstairs locations can now be entered exited correctly
  • Random Events can no longer spawn in the wilderness.
  • Fixed Yak Hide Armor crafting level requirements
  • Fixed Typo in Professor Oddenstein's dialogue

Backend Improvements

  • Add Remote Kicking support to Management Server
    • After manual verification a user can use Zanik to kick themselves from the server if they are stuck in a logged in state.
