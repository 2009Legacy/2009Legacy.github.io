Working Behind the scenes

  Summer
Website Developer
13-September-2021 10:55

  Hope everyone has had an amazing couple of weeks! We had quite the weekend with the host's machine needing repaired but we have all made it through and the server is now back up with some updates. The last couple of weeks I have been battling this virus so things have been a bit slow on my end when it came to website updates; however, the server has been getting a lot of behind the scenes work preparing for the 578 client.   The Fall season is returning as we get closer towards the end of September. The adventure bots will start to feel the Fall joy with some new dialogue. I was wondering as a community if any of you had any ideas for some Fall related things they can say.

Content Additions

  • Added proper music for the Ancient Guthix Temple.
  • Added a kill counter for Tormented demons replacing Bork in the ::stats command.
  • Leprechaun Larry's Farming Supplies shop added
Bug Fixes
  • Fixed the bunyip health formula and it's npc Id.
  • Corrected the damage reduction of the Tormented Demon's fire shield
  • Corrected the Tormented Demon's threshold for switching prayer type
  • Teleport is no longer blocked when spamming the teleport
  • Fly fishing rainbow fish now consumes stripy feathers instead of normal feathers.
  • Increased Tormented Demons aggro range
  • Added proper animations for Red Topaz Machete instead of just kicking enemies.
