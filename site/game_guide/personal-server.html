Singleplayer






For you new contributors lets get you up and running with how to set up your own personal server for 2009scape!

I just want to start off that I'm only moving this Readme to my personal website in order for organization purposes. Do not ask for help setting up your personal server because that's what this readme was for to begin with. It's a shortened version of the one on the official Gitlab repo. I am not a Windows Support person so I can ensure this will work on Linux and that's about it. If it works on Windows fantastic but this the official Readme has support for all Operating Systems.

This project has updated to Java 11 so you will need java 11.

For Debian based projects: sudo apt-get install openjdk-11-jre*
For Arch based projects: sudo pacman -S jdk11-openjdk
Frequently Asked Questions


What ports do I need to open up to be able to play with friends?

You just need to open 43594, 43595, and 5555. Keep in mind that only the server needs to have the open ports. Package the client and config file to send to your friends and you will be all set having your frineds connect.


How do I make myself an admin?

Just go to your phpmyadmin, go to your player file and set rights to 2.
